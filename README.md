# Kagi Vicinae Extension

A simple extension for [Vicinae](https://github.com/vicinaehq/vicinae) that provides web search suggestions for the [Kagi search engine](https://kagi.com). This extension is much simpler in implementation than the Kagi Raycast extension by not using API keys and just relying on the public autosuggest API. If you want more features, then you probably will want to use that extension.
